
# 🔄 Sincronização Local-to-Server via Git (Infraestrutura)

### 🚀 Objetivo do Sistema
Gerenciar o ciclo de vida e deploy dos projetos WordPress desenvolvidos localmente no Ubuntu diretamente para os servidores de produção dos clientes.

### 🛠️ Práticas de Controle
* **Versionamento Estrito:** Isolar arquivos do core do WordPress do código customizado desenvolvido pela agência.
* **Pontes de Automação:** Integrar scripts de automação que permitem que a própria IA valide se os arquivos modificados no diretório local estão prontos para commit, garantindo código limpo e livre de funções órfãs antes do deploy.
